# TechPoint E-commerce

Projeto de e-commerce de tecnologia desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades
- Busca dinâmica de produtos
- Filtro por categorias
- Layout responsivo
- Cards de produtos interativos
- Interface moderna inspirada em lojas online

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- Font Awesome

## Categorias disponíveis
- Smartphones
- Notebooks
- Fones
- Smartwatches
- Acessórios
