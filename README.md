## 🔍 Funcionalidades do Sistema

### 🔎 Pesquisa de Produtos

O usuário pode pesquisar produtos em tempo real através do campo de busca.

### 🗂️ Filtro por Categorias

Os produtos podem ser filtrados pelas categorias:

* Smartphones
* Notebooks
* Fones
* SmartWatches
* Acessórios

### 🛍️ Produtos Dinâmicos

Todos os produtos são renderizados dinamicamente utilizando JavaScript.
